hugo建站笔记
@ dongwt · Monday, Jul 13, 2020 · 1 minute read · Update at Jul 13, 2020

笔记

hugo new site myblog //建博客站

git clone https://github.com/xxxx //在github上下载一个主题

hugo server -t (theme) - -buildDrafts //启动本地web，可以查看写好的博客样式，theme为主题名

hugo new post/blog.md //建立markdown文件，也可以(hugo new blog.md)

hugo - -theme=（theme） - -baseUrl="https://dongwt96.github.io/” - -buildDrafts //将博客部署到github上

cd public //public文件夹为要上传的文件夹

git init //给文件夹建一个git站

git add . //将当前文件夹的所有内容加入暂存区

git commit -m “xxx” //将暂存区中的加入本地库，xxx为本次提交的标题

git remote add origin https://github.com/xxx //连接自己的github

git push -u origin master //将本地库中的文件推到github

只是重要步骤的笔记跟理解，具体细节看视频比较好

之后更新博客步骤,必要步骤为黑体

hugo new post/blog_0713.md //建立新的笔记本

hugo server -t (theme) - -buildDrafts //启动本地web，可以查看写好的博客

hugo - -theme=（theme） - -baseUrl="https://dongwt96.github.io/” - -buildDrafts //将博客部署到github上

cd public //public文件夹为要上传的文件夹

git add . //将当前文件夹的所有内容加入暂存区

git commit -m “xxx” //将暂存区中的加入本地库，xxx为本次提交的标题

git push -u origin master //将本地库中的文件推到github，有时候需要登录

其他问题：
博客摘要过长，在文章开头之后输入 ,中间没有空格
以前的博客要修改，直接在写好的博客里添加删除内容就可以（最开始特别纠结怎么更改，想着要不要把github里的内容全删了，再上传，还好试了下成功了）
